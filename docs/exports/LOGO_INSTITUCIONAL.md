# Logo institucional para exportaciones

Las exportaciones Excel, Word y PDF buscan automaticamente un logo institucional en alguna de estas rutas:

- app/assets/institution_logo.png
- app/assets/institution_logo.jpg
- app/assets/institution_logo.jpeg

Si el archivo existe, se incrusta en el encabezado del formato institucional.
Si no existe, el sistema usa un recuadro de marcador visual para no bloquear la exportacion.

Recomendacion:

- Colocar un PNG con fondo transparente y proporcion vertical similar al escudo institucional.
